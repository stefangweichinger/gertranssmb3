Besonderheiten und Vorteile

Das Samba Entwicklungsteam benötigt Feedback von Personen die ihre Samba Server sichern, also eine Sicherungskopie erstellt haben. Sie möchten wissen welche Art von Software Sie verwendet haben und wie die Rücksicherung funktioniert hat.

Wenn Sie im einzelnen Erfolg oder Fehler beim Backup von Samba Servern hatten, teilen Sie diese Information bitte mit dem Samba Team.
